## Meta

- charset 字符编码
- http-equiv=”X-UA-Compatible” 使用内核
- http-equiv="refresh" 重定向
- name=”description/keywords/author”
- name="robots" 检索查询
- name="viewport" 移动设备
  - content="width=device-width, initial-scale=1.0"
- webApp相关 略
